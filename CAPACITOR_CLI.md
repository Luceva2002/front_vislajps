# Comandi CLI di Capacitor (uso rapido)

- `npx cap doctor` – controlla setup e dipendenze.
- `npx cap add ios` / `npx cap add android` – aggiunge la piattaforma.
- `npx cap copy` – copia i web asset nella cartella nativa (non reinstalla i plugin).
- `npx cap sync` – copia asset e sincronizza i plugin nativi (equivale a copy + update plugin).
- `npx cap open ios` / `npx cap open android` – apre il progetto nativo in Xcode/Android Studio.
- `npx cap run ios` / `npx cap run android` – build e avvio su device/emulatore (richiede ambiente nativo configurato).
- `npx cap update` – aggiorna le versioni dei plugin nativi nei progetti nativi.
- `npx cap ls` – mostra le piattaforme installate.
- `npx cap --help` – elenco completo dei comandi e delle opzioni.
